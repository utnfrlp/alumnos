# alumnos
Repositorio donde los alumnos pueden:

1. Subir soluciones a problemas determinados e ir mejorándolas de forma comunitaria. Proponer desafíos o pedido de solución a problemas a la comunidad de desarrollo. (vía Pulls siempre)

2. Plantear soluciones alternativas o aplicaciones extra de ejercicios de las materias. (vía Pulls siempre)

3. Compartir sus proyectos de forma abierta. (libre)
